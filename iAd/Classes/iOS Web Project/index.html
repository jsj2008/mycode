Header bar

When this button is pressed, an iAd banner must be displayed at the top of the page. The banner should push the UIWebView down, so that the iAd banner and the contents of the UIWebView remain visible. The function should return true when the ad is displayed, or false when no ad is available.

When this button is pressed, the top iAd banner must be refreshed. The function should return true when the ad is refreshed, or false when no new ad is available.

This button simply hides the top iAd banner.

When this button is pressed, an iAd banner must be displayed at the bottom of the page. The banner should push the UIWebView up, so that the iAd banner and the contents of the UIWebView remain visible. The function should return true when the ad is displayed, or false when no ad is available.

When this button is pressed, the bottom iAd banner must be refreshed. The function should return true when the ad is refreshed, or false when no new ad is available.

This button simply hides the bottom iAd banner.

When this button is pressed, a full page ad must be displayed. This hides the entire UIWebView. The function should return true when the ad is displayed, or false when no ad is available.

This button simply hides the full page iAd banner.
